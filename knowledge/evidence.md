# Evidence & Reference Registry

## How to Use This File

### Before Citing
1. **ALWAYS check this file first** - 이미 검색한 reference인지 확인
2. 없으면 검색 후 이 파일에 등록
3. PDF 파일은 `knowledge/pdf/`에 저장

### New Reference Workflow
1. Search (PubMed, Google Scholar, etc.)
2. Verify paper exists and claim is accurate
3. Save PDF: `knowledge/pdf/author_year_keyword.pdf`
4. Register below with key points
5. Update "Used in" field when citing

### Reference Entry Format
```
### [#] Author et al., Year
- **File:** knowledge/pdf/filename.pdf (또는 "No PDF - abstract only")
- **Citation:** Full citation in target journal format
- **DOI:** 10.xxxx/xxxxx
- **PMID:** xxxxxxxx
- **Key Points:**
  - Point 1 (used in: Section, paragraph)
  - Point 2
- **Quotes (if any):** "exact quote" (page X)
- **Search Date:** YYYY-MM-DD
```
