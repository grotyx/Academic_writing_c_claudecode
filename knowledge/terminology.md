# Field Terminology Reference

> BESS/척추 수술 분야 표준 용어 사전
> Park SM 논문에서 일관되게 사용된 용어 기준 (2026-05-15)
> 새 논문 작성 전 확인 — 동의어 혼용 방지

---

## 수술 기법 명칭

| 사용할 용어 | 사용하지 않을 표현 | 비고 |
|-------------|-------------------|------|
| biportal endoscopic spine surgery (BESS) | biportal endoscopy, biportal endoscopic surgery | 기법 전체를 지칭하는 일반 명칭 |
| biportal endoscopic discectomy (BED) | biportal endoscopic lumbar discectomy | discectomy 특정 시 BED 약어 가능 |
| biportal endoscopic decompressive laminectomy | biportal endoscopic laminectomy | stenosis 수술 시 전체 명칭 |
| microscopic discectomy (MD) | open discectomy, conventional discectomy | 비교군 명칭 — "open microscopic" 불필요 |
| microscopic spinal surgery (MSS) | conventional microscopic surgery | pooled analysis에서 대조군 총칭 (Neurospine 2024) |
| unilateral laminotomy for bilateral decompression (ULBD) | bilateral decompression alone | 편측접근 양측 감압 시 |
| minimally invasive spine surgery (MISS) | MIS spine | 포괄 명칭 |

## 결과 지표 (Outcome Measures)

| 약어 | 전체 명칭 | 비고 |
|------|-----------|------|
| ODI | Oswestry Disability Index (version 2.0) | 10섹션, 각 0–5점, 백분율로 환산; MCID = 12.8점 |
| VAS | visual analog scale | 0–10; "NRS" 사용 금지 (별개 도구) |
| EQ-5D (EQ-5D-5L) | European Quality of Life-5 Dimensions (five-level) | −0.066 to 1.000; 높을수록 좋음 |
| painDETECT | painDETECT questionnaire | −1 to 38점; >19 = 신경병증성 통증 가능성 높음 |
| CCI | Charlson Comorbidity Index | 동반질환 지수 |
| ASA PS | American Society of Anesthesiologists physical status | 수술 위험 분류 |
| POSAS | Patient and Observer Scar Assessment Scale | 반흔 평가; 6 parameters, 1–10 (6=정상, 60=최악) |
| PRO / PROMs | patient-reported outcome (measures) | "patient questionnaire" 아님 |
| CPK ratio | creatine phosphokinase ratio | postop day 1 CPK / preop CPK — 근육 손상 대리 지표 |

## 근육 손상 지표: CPK 관련 용어

> 자주 틀리는 항목 — 주의

| ✅ 사용할 용어 | ❌ 사용 금지 | 이유 |
|--------------|-------------|------|
| creatine phosphokinase (CPK) | creatinine phosphokinase | creatinine ≠ creatine |
| serum CPK level | serum CK level | Park SM 논문에서 CPK 일관 사용 |
| CPK ratio | CPK index | ratio는 postop/preop 비율로 정의됨 |

## 분류 체계

| 명칭 | 적용 대상 | 비고 |
|------|-----------|------|
| Schizas classification (Grade A–D) | 요추관 협착 중증도 | Grade ≥B = definite central stenosis (포함 기준) |
| Meyerding classification (Grade I–IV) | 척추전방전위증 | Grade ≥II = 제외 기준 (RCT) |
| Cobb angle | 척추측만증 | >20° = 제외 기준 |
| Charlson Comorbidity Index | 동반질환 | 연속형 또는 이분형으로 보고 |
| ASA physical status (1–5) | 마취 위험 | 2 또는 2.0(0.X) 형태로 보고 |

## 연구 설계 용어

| 사용할 용어 | 혼용 주의 | 비고 |
|-------------|-----------|------|
| assessor-blind | double-blind | 수술 RCT는 구조적으로 double-blind 불가 |
| single-blind | — | 참여자가 배정을 알 때 |
| modified intention-to-treat (mITT) | ITT | 무작위 배정 후 수술 진행한 환자 포함 |
| per-protocol (PP) | — | 프로토콜 위반 없는 환자만 포함; 민감도 분석 |
| non-inferiority trial | equivalence trial | 일방적 가설 (한쪽이 열등하지 않음) |
| equivalence trial | — | 양방향 가설 |
| non-inferiority margin | — | ODI 기준 12.8점 (MCID, Park SM RCT 일관) |
| parallel-group trial | — | 두 군이 동시에 진행 |
| multicenter / multicentre | multi-center | 영미식 혼용 가능; 저널별 일관성 유지 |
| computer-generated randomization | random allocation | 컴퓨터 난수 생성 강조 |
| block randomization (block size of 4) | — | Park SM RCT 표준 |
| allocation concealment | — | 배정 은닉; eCRF 또는 sealed envelopes |

## 통계 용어

| 사용할 표현 | 사용 금지 | 비고 |
|-------------|-----------|------|
| mean (SD) | mean ± SD | 표준 표기; SD는 괄호 안 |
| median (IQR) | median ± IQR | 비정규분포 시 |
| 95% confidence interval (CI) | 95% confidence limits | CI 약어 사용 |
| p = 0.023 | p < 0.05, p = NS | 정확한 p값 기재 권장 |
| two-sided p < 0.05 | — | 통계적 유의수준 기술 표준 |
| Shapiro-Wilk test | Kolmogorov-Smirnov | 정규성 검정 (Park SM 논문 표준) |
| Fisher's exact test | Chi-square (소표본) | 범주형 소표본 |
| Mann-Whitney U test | Wilcoxon rank-sum test | 동일 검정; Mann-Whitney 선호 |
| linear mixed model | repeated-measures ANOVA | 반복 측정 연속형 결과 (Park SM RCT 표준) |
| last observation carried forward (LOCF) | — | 결측 처리 방법 |
| Stata/MP | — | Park SM 논문 표준 소프트웨어 |

## 합병증/이상반응 용어

| 사용할 용어 | 비고 |
|-------------|------|
| adverse events | complications보다 포괄적 — 안전성 분석 시 권장 |
| complications | 수술 기술적 문제 기술 시 |
| dural tear | durotomy와 구별; dural tear = 의도치 않은 파열 |
| durotomy | 의도적 또는 비의도적 경막 절개 |
| epidural hematoma | subdural hematoma와 구별 |
| wound dehiscence | 창상 열개 — Park SM 논문에서 key outcome |
| nerve root injury | radicular injury 아님 |
| recurrent disc herniation | re-herniation 아님 |
| incomplete decompression | inadequate decompression 아님 |
| reoperation | revision surgery; re-operation (hyphen 없이) |

## 병원 및 임상 용어

| 사용할 표현 | 비고 |
|-------------|------|
| participants | patients (RCT 논문) — 임상시험 참여자에는 participants 권장 |
| patients | 임상 연구, 후향적 연구에서 사용 가능 |
| tertiary institution / tertiary referral center | 3차 병원 |
| orthopedic / orthopaedic | 저널별 일관성 유지 — 영미식 차이 (BJJ = orthopaedic, Spine J = Orthopedic 혼용) |
| lumbar disc herniation | herniated nucleus pulposus, HNP | 전자가 더 포괄적으로 쓰임 |
| lumbar spinal stenosis | LSS, lumbar stenosis | 풀네임 선호 (첫 언급 시) |
| radiating pain | radiculopathy | 증상 기술 시 radiating pain; 진단은 radiculopathy |
| lower extremity | lower limb | 미국식 표현 선호 |

## ⚠️ 자주 틀리는 항목 요약

| ❌ 잘못된 표현 | ✅ 올바른 표현 |
|--------------|--------------|
| creatinine phosphokinase | creatine phosphokinase (CPK) |
| biportal endoscopy (수술명으로) | biportal endoscopic spine surgery (BESS) |
| double-blind (수술 RCT) | assessor-blind |
| p < 0.05 (결과 보고 시) | p = [정확한 값] |
| re-herniation | recurrent disc herniation |
| NRS | VAS (별개 도구 — 혼용 금지) |
| MIS (단독 약어) | MISS (minimally invasive spine surgery) |
| open surgery | microscopic surgery / microscopic discectomy |
| orthopaedic vs orthopedic | 저널별 통일 (혼용 금지) |
