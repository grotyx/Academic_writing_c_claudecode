# Terminology Registry

> Field-standard terminology for manuscript drafting and lint checks.
> This file belongs under `Style/` because it controls writing style and vocabulary, not citation evidence.
> Confirm project-specific terms in `drafts/draft_plan.md` before drafting.

---

## How to Use

- Use the `Preferred Term` exactly unless the target journal requires otherwise.
- Do not use terms listed under `Forbidden Terms`.
- Define abbreviations once at first use in the main text. Abstract is treated independently.
- If two terms are both acceptable, choose one in the draft plan and use it consistently.
- Add project-specific terminology before drafting when a new procedure, outcome, disease entity, or group name is introduced.
- Do not substitute one measurement instrument for another. If the dataset measured NRS, report NRS; if it measured VAS, report VAS.
- If the target journal uses British spelling, record that decision in `drafts/draft_plan.md` before drafting.

---

## Scope and Override Rules

| Rule | Default | Override When |
|---|---|---|
| Term source | This registry controls preferred vocabulary | Target journal instructions conflict |
| Measurement names | Use the exact instrument measured | None; never convert VAS to NRS or NRS to VAS in text |
| Procedure names | Use specific procedure terms | A broader pooled category is preplanned |
| Spelling | US spelling default | Journal requires British spelling |
| Abbreviations | Define once at first use | Very common journal-approved abbreviations may remain undefined only if allowed |
| Forbidden terms | Replace before submission | A direct quotation or reviewer comment requires the original term |

## High-Priority Forbidden Terms

| Forbidden Term | Preferred Term | Reason |
|---|---|---|
| creatinine phosphokinase | creatine phosphokinase (CPK) | Creatinine is incorrect |
| biportal endoscopy | biportal endoscopic spine surgery (BESS) | Procedure naming consistency |
| double-blind | assessor-blind | Surgical RCTs are structurally not double-blind unless specifically justified |
| p = NS | exact p-value or p < 0.001 | Non-standard reporting |
| p = 0.000 | p < 0.001 | P-values are not zero |
| re-herniation | recurrent disc herniation | Preferred complication terminology |
| substituting NRS for VAS or VAS for NRS | report the measured instrument exactly | Separate instruments; do not mix |
| MIS | minimally invasive spine surgery (MISS) | Avoid ambiguous standalone abbreviation |
| open surgery | microscopic surgery or microscopic discectomy | Use comparator-specific term |
| compared to | compared with | AMA-style preference for comparisons |
| equivalent | noninferior or not superior, if that was the actual design/result | Equivalence requires a specific two-sided design |
| AI replacement | AI-assisted draft or clinician-reviewed output | Clinical AI should not be framed as replacing expert judgment unless proven |
